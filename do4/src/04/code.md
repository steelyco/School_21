**2xx (Успешные):**

200 OK - стандартный успешный ответ

201 Created - ресурс успешно создан

400 Bad Request - сервер не понял запрос из-за неверного синтаксиса

**4xx (Ошибки клиента):**

401 Unauthorized - требуется аутентификация

403 Forbidden - доступ запрещен

404 Not Found - ресурс не найден

**5xx (Ошибки сервера):**

500 Internal Server Error - общая ошибка сервера

502 Bad Gateway - сервер-шлюз получил недопустимый ответ

503 Service Unavailable - сервер временно не доступен

504 Gateway Timeout - сервер-шлюз не дождался ответа
